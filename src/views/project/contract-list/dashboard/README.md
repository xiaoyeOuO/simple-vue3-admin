# 项目看板组件说明

## 功能概述

`dashboardPane.vue` 是一个基于ECharts的数据可视化组件，用于展示项目的关键指标和趋势数据。

## 图表功能

### 1. 缺陷数量变化趋势图
- **数据维度**：过去30天的缺陷数量变化
- **图表类型**：支持折线图、柱状图、饼图切换
- **颜色主题**：红色系（#f56c6c）

### 2. 任务完成数量变化趋势图
- **数据维度**：过去30天的任务完成数量变化
- **图表类型**：支持折线图、柱状图、饼图切换
- **颜色主题**：蓝色系（#409eff）

### 3. 实际工时变化趋势图
- **数据维度**：过去30天的实际工时消耗
- **图表类型**：支持折线图、柱状图、饼图切换
- **颜色主题**：绿色系（#67c23a）

## 交互特性

### 时间范围选择
顶部提供时间范围选择器：
- 📅 **近7天**：查看最近一周数据
- 📅 **近15天**：查看最近半个月数据
- 📅 **近30天**：查看最近一个月数据（默认）
- 📅 **近90天**：查看最近三个月数据
- 📅 **自定义日期范围**：通过日期选择器选择任意时间段
- 🔄 **刷新按钮**：手动刷新最新数据

### 图表类型切换
每个图表右上角提供图表类型切换按钮：
- 📈 折线图：展示趋势变化
- 📊 柱状图：对比每日数据
- 🥧 饼图：展示数据占比

### 响应式设计
- 桌面端：三列网格布局
- 移动端：单列布局，自适应屏幕
- 图表自动调整大小

### 数据提示
- 鼠标悬停显示详细数据
- 饼图显示百分比
- 折线图/柱状图显示具体数值

## 技术实现

### 技术栈
- **图表库**：ECharts 6.0.0
- **UI框架**：Element Plus
- **响应式**：CSS Grid + Flex
- **状态管理**：Vue 3 Composition API

### 数据结构
```javascript
{
  dates: ['01-01', '01-02', ...],     // 日期标签
  bugData: [5, 8, 3, ...],            // 缺陷数量
  taskData: [12, 15, 8, ...],         // 任务完成数量
  workHourData: [25, 30, 18, ...]     // 实际工时（小时）
}
```

### 组件API

#### Props
- `bugsData` - 缺陷数据数组
- `tasksData` - 任务数据数组
- `projectId` - 项目ID

#### 暴露方法
- `updateCharts()` - 手动更新图表数据
- `cleanup()` - 清理图表实例

## 使用方式

```vue
<dashboard-pane 
  :bugs-data="bugsData"
  :tasks-data="tasksData"
  :project-id="projectId"
/>
```
